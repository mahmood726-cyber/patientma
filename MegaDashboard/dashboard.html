About This Dashboard

The Patient-Centred CV Meta-Analysis MegaDashboard synthesises 12 analytical angles across 65,383 cardiovascular trials registered on ClinicalTrials.gov. It examines what trials measure, who they include, where and how they report, and how these patterns have evolved over time.

Each of the four tabs corresponds to one manuscript in a linked multi-paper series. All analyses use programmatically extracted data with reproducible pipelines.

Data source: ClinicalTrials.gov AACT database (Aggregate Analysis of ClinicalTrials.gov). Trials classified as "cardiovascular" based on condition mesh terms and keyword matching.

Gap Score: A composite metric (0-1) measuring the extent to which a trial's primary outcomes align with patient-relevant core outcome sets. Higher values indicate greater patient-centricity.

Generalisability Index (GI): A multidimensional score (0-100) reflecting how well trial eligibility criteria match real-world patient demographics across age, sex, comorbidity, functional status, and population representation dimensions.

Built with vanilla HTML/CSS/JS (no dependencies). Canvas-based charts for maximum portability.
